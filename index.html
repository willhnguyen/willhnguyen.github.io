UCD Eats

Senior Design (Jan. 2015 â€“ June 2015)

UCD Eats is a mobile app that provides nutrition information for the foods available at the UC Davis Coffee House and allows users to log the information to Fitbit. Users are also given the option to modify various food items based on their order via the Build-Your-Own feature.

This app was created using the Ionic Framework and Facebook's Parse database. Ionic is a web framework that packages HTML, CSS, and JavaScript code into an iOS and Android app. This allowed us to focus on the content without worrying too much about the UI and UX.

My role involved the front-end development of the app. I created and modified the views, search modal, and the Build-Your-Own feature.

Technology Used: Ionic Framework, Facebook Parse, HTML, CSS, JavaScript, AngularJS
